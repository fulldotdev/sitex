Every component installs with one command — `pnpm dlx shadcn@latest add https://sitex.full.dev/r/{name}.json` — and each page shows verified usage and props.

| Component                               | Description                                                                                  |
| --------------------------------------- | -------------------------------------------------------------------------------------------- |
| [`banner`](/docs/ui/banner)             | Dismissable site-wide announcement banner.                                                   |
| [`code`](/docs/ui/code)                 | Code block with shiki highlighting and a copy button; drop-in MDX `pre` replacement.         |
| [`gradient`](/docs/ui/gradient)         | Decorative gradient surface for hero and section backgrounds.                                |
| [`header`](/docs/ui/header)             | Site header with container and grouping primitives.                                          |
| [`layout`](/docs/ui/layout)             | Document primitives for Sitex layouts: html, head with SEO meta and JSON-LD, body, and main. |
| [`marquee`](/docs/ui/marquee)           | Continuously scrolling strip for logos and quotes.                                           |
| [`price`](/docs/ui/price)               | Locale-aware price display with sale and discount formatting.                                |
| [`rating`](/docs/ui/rating)             | Star rating display.                                                                         |
| [`section`](/docs/ui/section)           | Page section with consistent vertical rhythm and container.                                  |
| [`theme-toggle`](/docs/ui/theme-toggle) | Light/dark theme provider and toggle button.                                                 |
| [`tile`](/docs/ui/tile)                 | Card-like tile and tile group for feature grids.                                             |
| [`toc`](/docs/ui/toc)                   | Table of contents navigation for page headings.                                              |
| [`typography`](/docs/ui/typography)     | Prose typography primitives for MDX content.                                                 |
