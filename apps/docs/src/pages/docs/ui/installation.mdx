Sitex UI components assume a Sitex app with Tailwind. Add Tailwind first if you have not:

```bash
pnpm add tailwindcss && pnpm add -D @tailwindcss/vite
```

```ts
// vite.config.ts
import tailwindcss from "@tailwindcss/vite"

export default defineConfig({
  plugins: [react(), sitex(), tailwindcss()],
})
```

```css
/* src/index.css */
@import "tailwindcss";
```

## shadcn

Initialize shadcn once — it creates `components.json` and `src/lib/utils.ts`:

```bash
pnpm dlx shadcn@latest init
```

## Add Components

Install any component from the Sitex registry. The source lands in `src/components/ui` and dependencies install automatically:

```bash
pnpm dlx shadcn@latest add https://sitex.full.dev/r/section.json
```

Sitex UI components compose with regular [shadcn/ui](https://ui.shadcn.com) components — mix both in the same app.
